## Rafael Woronkoff's Portfolio

This repository contains the source code for the my personal portfolio website. This is a multi-page static website built with HTML and CSS to showcase his professional profile, projects, and contact information.

## Overview
The portfolio is designed to be a clean and professional representation of Rafael's skills as a developer. It provides an overview of his background, education, and showcases a variety of projects he has worked on.

## Pages

The website is organized into the following pages:

- Início (Home): (index.html) The main landing page that offers a brief introduction and welcome message.
- Sobre (About): (sobre.html) Contains information about Rafael, including his passions, interests, and hobbies such as aviation and game programming.
- Formação (Education): (formacao.html) Details his educational background, including his Bachelor's degree in Software Engineering and language proficiencies.
- Portfólio (Portfolio): (portfolio.html) A gallery of projects with descriptions and links to each.
- Contato (Contact): (contato.html) A page with a contact form and other contact details like email and LinkedIn.

## Features

- Responsive Design: The site is fully responsive and works across different devices, including desktops, tablets, and mobile phones.
- Clean and Modern UI: A simple and intuitive user interface styled with custom CSS.
- Project Showcase: A dedicated section to display projects with links for live viewing.
- Multi-language Support: Information about language proficiency in Portuguese, English, and German.

## Technologies Used
- HTML5
- CSS3

## Projects Featured
The portfolio section highlights the following projects:

- Meu Linktree Project: An HTML/CSS project that mimics a Linktree page.
- Tire uma Carta (Draw a Card): A simple card game built with Python.
- Sistema de Pedido de Comida (Food Order System): A basic food order management system in Python.
- Pedra, Papel, Tesoura (Rock, Paper, Scissors): The classic game implemented in Python.
- Biblioteca (Library): A simple library management system in Python.

## How to Run
To view the website locally, follow these steps:

1. Clone the repository:

        git clone https://github.com/woronkoff/rafaelworonkoff-portfolio.git

2. Navigate to the project directory:

        cd rafaelworonkoff-portfolio

3. Open the index.html file in your web browser.
